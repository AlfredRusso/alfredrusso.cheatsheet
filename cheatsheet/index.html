HTML TABLE REFERENCES

Table tags

Tag Name Description
table Table The wrapper element for all HTML tables.
thead Table Head The set of rows defining the column headers in a table.
tbody Table Body The set of rows containing actual table data
tr Table Row The table row container
td Table Data The Table row content
tfoot Table Footer The set of rows defining the footer in a table.

Table Attributes

Attribute Name Descritpion
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span

TYPOGRAPHY IN CSS

Typography is the art of arranging text on a page.

CSS Description
font-weight Makes text appear in any number of weights
font-style Makes text appear in other styles, such as italic
line-height Modifies the vertical spacing between lines of text
like so
Serif Serif fonts have extra details on the ends of each letter
Sans-serif Sans-Serif fonts do not
Word-spacing Changes how far apart individual words are
Letter-spacing Changes how far apart individual letters are
Text-align Changes the horizontal alignment of text
font-family Changes the font of the text
